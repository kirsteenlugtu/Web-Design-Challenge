Weather vs Latitude

The purpose of this project is to explore changes in weather across cities with different latitude. For these analyses weather data was pulled from the OpenWeatherMap API of over 500 cities that corresponded to a set of randomly chosen latitude values. Temperature, cloudiness, humidity and wind speed were collected for the dataset.

Matplotlib was used to plot the weather data across the different latitude values. This site provides figures of the plots and any observations made from the analyses as well as a data table of the data set that was used for these analyses.
