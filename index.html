Weather vs Latitude

The purpose of this project is to explore changes in weather across cities with different latitude. For these analyses weather was pulled from the OpenWeatherMap API of over 500 cities randomly chosen from randomly chosen latitudes. Temperature, cloudiness, humidity and wind speed were collected for the dataset. Matplotlib was used to plot the weather data across the different latitude values. In this site you can find figures of the plots and any observations made from the analyses. You can also find a data table of the data set that was used for these analyses.
